# Frontend Mentor - Blog Preview Card

## Overview
This is my solution to the **Blog Preview Card** challenge from [Frontend Mentor](https://www.frontendmentor.io).  
The goal of this challenge is to build a responsive blog preview card that closely matches the provided design, with a focus on layout, typography, and hover states.

## Built With
- HTML5  
- CSS3  

This project was built using **pure HTML and CSS**, without JavaScript or frameworks.

---

## My Process
1. Structured the layout using semantic HTML.
2. Styled the card with CSS, focusing on:
   - Spacing and alignment
   - Typography and font weights
   - Image styling and border radius
   - Hover states for interactivity
3. Tested responsiveness to ensure the card looks good on different screen sizes.

---

## What I Learned
- How to better control spacing and alignment in small UI components.
- How to match design specifications using CSS variables.
- The importance of hover states in improving user experience.
- Improved confidence in converting designs into clean HTML & CSS.
